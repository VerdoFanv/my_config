# MYSQL

## COMMANDS
```
mysql -u root
create database <name>;
show databases;
drop database <name>;
use <db name>;
show tables;
create table <table name>;
desc <table name>;
drop table <table name>;
alter table <name> add
alter table <name> modify
alter table <name> change 
alter table <name> drop <field name>
alter table <name> rename <new name>
```
